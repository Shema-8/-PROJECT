SHINE YOUNG – Cultural Learning Platform

 **Overview**

SHINE YOUNG is a full-stack web application designed to preserve and promote cultural knowledge among students. The platform allows users to explore traditional stories, proverbs, cultural media, and school-based cultural activities in an interactive and modern environment.

This project demonstrates the application of web development concepts including frontend design, backend development, and user interaction features.

**Objectives**

The main objective of this project is to create a digital solution that helps young learners reconnect with their cultural heritage. The platform aims to make cultural education more accessible, engaging, and interactive through technology.

** Problem**

Cultural knowledge is gradually being lost due to globalization and reduced traditional teaching methods. Many students lack access to structured cultural learning resources.

**Solution**

SHINE YOUNG provides a centralized platform where users can easily access and interact with cultural content. It enhances learning through features such as bookmarking, search, and personalized user experience.

**Content Management**

* View stories, proverbs, media, and school culture
* Dynamic content loaded from backend

** Architecture**

The application follows a client-server architecture:

* Frontend handles UI and user interaction
* Backend serves APIs and static files
* Data is fetched dynamically via HTTP requests
